연산자
======
1.연산자
-----
### 1_연산자와 피연산자
* 연산자 : +-등 연산을 수행하는 기호
* 피연산자 : 연산자의 작업대상(변수,상수,리터럴,수식)

### 2_식과 대입연산자
* 식 = 연산자와 피연산자의 조합
* 식을 평가한다 = 식을 계산하는것 

**대입연산자**   
=을 통하여 식의 값을 저장한다.   
ex)
```
y= 5+x;
```
x와 5를 더한값을 y에 저장

### 3_연산자의 종류

|연산자|기호|설명|
|---|---|---|
|산술 연산자| + - * / % << >>|(사칙연산과 나머지연산(%))|
|비교 연산자| > < >= <= == != |크고작음을 비교|
|논리 연산자| && || ! & ^ ~|그리고 또는 으로 조건 연결|
|대입 연산자| = |우변의 값을 좌변에 저장|
|기타|(type) ?: instanceof |형변환 연산자, 삼항 연산자, instanceof 연산자|
